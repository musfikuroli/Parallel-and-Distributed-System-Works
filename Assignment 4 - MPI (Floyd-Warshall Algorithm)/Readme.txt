Important Commands for Running The Programs:
	
	1. For Generating Random input.txt:
		“make generateInput”
	
	2. For Running Serial Program:
		“make serialFloyd”
		
	3. For Running The Parallel Program:
		“make parallelFloyd”
		
	4. For Running The Pipeline Program:
		“make pipelineFloyd”
		
	5. For Generating & Showing The Statistics (Speedup Graphs)
		“cd Stat && python3 graph.py”
