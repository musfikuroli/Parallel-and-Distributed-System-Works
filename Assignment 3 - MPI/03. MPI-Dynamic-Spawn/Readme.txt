Using Makefile:-

1. For Cleaning Executables:

	"make clean"
	
----------------------------------------------------------------------------------------

2. For Compiling through mpicc:
 
	"make all"

----------------------------------------------------------------------------------------

3. For Running The Program.

	"make run"
	
	and then Give No of Child/Worker to be Spawned: 4/5/8 etc. and Press Enter.
----------------------------------------------------------------------------------------
----------------------------------------------------------------------------------------



----------------------------------------------------------------------------------------
----------------------------------------------------------------------------------------
Manually Run (Without Makefile):-

1. Compile Master:
	mpicc -o master master.c

2. Compile Worker:
	mpicc -o worker worker.c

3. Run Master:
	mpirun --oversubscribe -np 1 ./master
	
	and then Give No of Child/Worker to be Spawned: 4/5/8 etc. and Press Enter.
